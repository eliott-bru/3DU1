Il s'agit d'un jeu d'aventure dans lequel le joueur doit explorer un donjon afin de découvrir le trésor qui s'y cache.

J'ai choisi de réaliser tous les assets moi-même, ce qui a été très chronophage.

Vous pouvez faire pause en appuyant sur Echap.











------------------------
	SOLUCE:
------------------------

1. Dans la première salle, il suffit d'attraper une torche et d'allumer le flambeau avec la touche d'interraction.

2. Dans la seconde salle, si vous n'arrivez pas le parcours dans le temps imparti, il y a un sol invisible en longeant le mur gauche.

3. Dans la salle avec la tête de singe, pour ouvrir la porte il faut réussir les 2 épreuves situées dans les salles à droite et à gauche de celle-ci.

4. Dans la salle à gauche de la porte avec la tête de singe, il y a une boule de pierre qui tombe lorsque vous vous approchez du sommet. Pas de miracle, vous allez plus vite que la boule.

5. Dans la salle à droite de la porte avec la tête de singe, il faut amener le rayon lumineux sur le crystal au milieu de la pièce, pour cela il faut tourner les mirroirs.
Depuis l'état de départ et dans l'ordre où le rayon touche les mirroirs, il faut les tourner 3, 3, 0, 2, 1 fois.

6. Dans la dernière salle, la salle du trésor il suffit d'ouvrir le coffre et d'interragir avec le saphir.
